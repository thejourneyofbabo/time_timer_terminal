# Time Timer

A circular countdown timer for the terminal.

## Usage

```bash
./timetimer 15m      # 15 minutes
./timetimer 30s      # 30 seconds
./timetimer 1h       # 1 hour
./timetimer 1h30m    # 1 hour 30 minutes
```

## Installation

```bash
chmod +x timetimer
sudo cp timetimer /usr/local/bin/
```

Then use it from anywhere:

```bash
timetimer 15m
```
